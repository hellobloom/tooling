# ESLint Config

Bloom's ESLint Configurations

## Installation

```
npm install --save-dev @bloomprotocol/eslint-config
```

## Usage

```js
// .eslintrc.js

module.exports = {
  extends: ['@bloomprotocol/eslint-config'],
}
```

Or for React support:

```js
// .eslintrc.js

module.exports = {
  extends: ['@bloomprotocol/eslint-config/react'],
}
```
